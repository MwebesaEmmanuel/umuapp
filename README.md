# UMU App (Mobile + Backend)

Monorepo for a cross‑platform university mobile app (students, staff, admins, visitors) and its backend APIs.

## Recommended stack

- Mobile: Flutter (Android + iOS)
- Backend: Python (FastAPI)
- Database: PostgreSQL (production), SQLite (local dev)

## What’s included so far (backend MVP skeleton)

- Auth: university email signup + login + roles (student/staff/admin)
- Content: announcements + events
- Community chat: rooms + message history + WebSocket chat
- Digital ID + attendance: signed QR token + session check-in
- Campus navigation: office directory with coordinates
- Support: helpdesk tickets + hostel maintenance issues
